# AirfieldCam Pi

Raspberry Pi relay software for AirfieldCam.

## Installation

On a fresh Raspberry Pi OS install, run:

```bash
curl -fsSL https://raw.githubusercontent.com/airfieldcam/pi/main/install.sh | bash
```

Then open a browser on your local network and go to:

http://<pi-ip-address>:8080
You will be prompted to set a password on first visit, then you can add and configure your cameras.

## Updating

To update to the latest version:

```bash
curl -fsSL https://raw.githubusercontent.com/airfieldcam/pi/main/update.sh | bash
```

## Requirements

- Raspberry Pi (any model with WiFi or Ethernet)
- Raspberry Pi OS (32-bit or 64-bit)
- Internet connection
- Cameras accessible on your local network
